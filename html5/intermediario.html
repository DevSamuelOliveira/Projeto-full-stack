Tags para colocar textos em itálico e negrito

Strong e B para negrito

Existem duas formas de se colocar um texto em negrito com html5 a tag <b> e <strong> pórem a forma correta é a <strong> ela diz para os leitores de textos que é uma palavra em negrito fazendo assim uma técnica de SEO para os buscadores do google. Se for usar a tag a antes fazer com css

Texto em negrito com <strong> diz para o buscador que é um texto em negrito
Texto em negrito com <b> apénas um estilizador que coloca o texto em negrito

Em e I para itálico

O mesmo que acontece com strong e b, acontece com em e i
o em é usado como técnica de SEO para os buscadores do google saber que é uma palavra que deve ser lida com ênfase e tag i apenas estiliza o texto, novamente repito antes fazer com o css
